3. Trabajaremos con la base WDI del Banco Mundial

World Development Indicators contiene miles de indicadores para diversos países, regiones y años (desde 1960 donde hay información).

Los países vienen codificados en ISO 3166-1 alpha-3: un estándar internacional de tres letras que identifica de manera única a cada país (MEX, HND, USA…). Los indicadores del WDI son codificados por el Banco Mundial.

Si ya descargaste los datos del sitio del Banco Mundial y los tienes en un .dta:

3.2 Opción B (la más pro): Descargar directo con la API 🚀

Esta es la forma recomendada porque no necesitas archivos locales y el código es completamente reproducible.

Tip¿Qué es una API?

Es una conexión directa entre tu programa y la base de datos del Banco Mundial. En vez de descargar un Excel, le dices exactamente qué indicadores, qué países y qué años quieres, y el Banco Mundial te los manda al momento.

4. Limpieza de nombres de variables

Cuando lees datos de Stata o del WDI, los nombres de las columnas pueden ser largos, con puntos, mayúsculas o caracteres raros. Aquí aprendemos a limpiarlos.

Advertencia🟡 Ojo con las etiquetas

Si renombras usando etiquetas automáticamente, pueden aparecer espacios o caracteres especiales que rompen el código. La opción 2 (manual) es más segura.

  • 🟦 R
  • 🟡 Stata
  • 🐍 Python
# Limpieza de nombres y creación de variable peers
wdi1 <- wdi_raw %>%
  janitor::clean_names() %>%           # pasa todo a snake_case
  rename(
    countrycode = iso3c,
    countryname = country
  ) %>%
  mutate(
    peers = if_else(countrycode %in% peers, 1, 0)
  )

# Verificar
table(wdi1$peers)

# 🧪 Extra: ver qué etiqueta tiene cada variable (útil con .dta)
data.frame(
  variable = names(wdi_raw),
  label    = sapply(wdi_raw, function(x) attr(x, "label"))
)
* Renombrar variables manualmente (más seguro)
rename countrycode iso3c     // ajusta según tu base

* Crear variable peers
gen peers = 0
foreach p of global peers {
    replace peers = 1 if iso3c == "`p'"
}

5.2 Tasa de crecimiento del PIB per cápita

Para calcular la tasa de crecimiento anual necesitamos comparar cada año con el anterior. Eso se llama primera diferencia logarítmica (o simplemente variación porcentual año a año).

  • 🟦 R
  • 🟡 Stata
  • 🐍 Python
wdi2 <- wdi1 %>%
  arrange(countrycode, year) %>%
  group_by(countrycode) %>%
  mutate(
    gdp_pc_growth = 100 * (ny_gdp_pcap_kd / lag(ny_gdp_pcap_kd) - 1)
  ) %>%
  ungroup()

grafico2 <- wdi2 %>%
  filter(countrycode == pais, year >= 1960, year <= 2023)

ggplot(grafico2, aes(x = year, y = gdp_pc_growth)) +
  geom_line(color = "#2C3E50", linewidth = 1.2) +
  geom_point(color = "#2980B9", size = 2, alpha = 0.7) +
  theme_minimal(base_size = 14) +
  labs(
    title    = "📈 Tasa de crecimiento del PIB per cápita en Honduras",
    subtitle = "Cálculo a partir de dólares constantes",
    x = "Año", y = "Crecimiento anual (%)",
    caption  = "Fuente: Banco Mundial – WDI"
  ) +
  scale_y_continuous(labels = scales::percent_format(scale = 1)) +
  scale_x_continuous(breaks = seq(1960, 2020, by = 10))

5.4 Empleo por sector (gráfico de barras apiladas)

  • 🟦 R
  • 🟡 Stata
  • 🐍 Python
anios <- c(1991, 1995, 2000, 2005, 2010, 2015, 2020)

empleo_sector <- wdi1 %>%
  filter(countrycode == pais, year %in% anios) %>%
  select(year, sl_agr_empl_zs, sl_ind_empl_zs, sl_srv_empl_zs) %>%
  pivot_longer(
    cols      = starts_with("sl_"),
    names_to  = "sector",
    values_to = "empleo"
  ) %>%
  mutate(
    sector = case_when(
      sector == "sl_agr_empl_zs" ~ "Agricultura",
      sector == "sl_ind_empl_zs" ~ "Industria",
      sector == "sl_srv_empl_zs" ~ "Servicios"
    )
  )

ggplot(empleo_sector, aes(x = factor(year), y = empleo, fill = sector)) +
  geom_bar(stat = "identity") +
  geom_text(
    aes(label = round(empleo, 1)),
    position = position_stack(vjust = 0.5),
    color = "white", size = 3.5
  ) +
  scale_fill_manual(values = c(
    "Agricultura" = "#1F77B4",
    "Industria"   = "#AAAAAA",
    "Servicios"   = "#C0392B"
  )) +
  labs(
    title    = "Distribución del empleo por sector en Honduras",
    subtitle = "Años seleccionados (valores en porcentaje)",
    x = "Año", y = "Porcentaje de empleo", fill = "Sector",
    caption  = "Fuente: WDI – Banco Mundial"
  ) +
  theme_minimal(base_size = 14)

# 4. Limpieza de nombres de variables

Cuando lees datos de Stata o del WDI, los nombres de las columnas pueden ser largos, con puntos, mayúsculas o caracteres raros. Aquí aprendemos a limpiarlos.

::: {.callout-warning}
## 🟡 Ojo con las etiquetas
Si renombras usando etiquetas automáticamente, pueden aparecer espacios o caracteres especiales que rompen el código. La opción 2 (manual) es más segura.
:::

::: {.panel-tabset}

## 🟦 R

```r
# Limpieza de nombres y creación de variable peers
wdi1 <- wdi_raw %>%
  janitor::clean_names() %>%           # pasa todo a snake_case
  rename(
    countrycode = iso3c,
    countryname = country
  ) %>%
  mutate(
    peers = if_else(countrycode %in% peers, 1, 0)
  )

# Verificar
table(wdi1$peers)

# 🧪 Extra: ver qué etiqueta tiene cada variable (útil con .dta)
data.frame(
  variable = names(wdi_raw),
  label    = sapply(wdi_raw, function(x) attr(x, "label"))
)
```

## 🟡 Stata

```stata
* Renombrar variables manualmente (más seguro)
rename countrycode iso3c     // ajusta según tu base

* Crear variable peers
gen peers = 0
foreach p of global peers {
    replace peers = 1 if iso3c == "`p'"
}

tab peers
```

## 🐍 Python

```python
import numpy as np

# Limpiar nombres: minúsculas y guiones bajos
wdi_raw.columns = (
    wdi_raw.columns
    .str.lower()
    .str.replace(".", "_", regex=False)
    .str.replace(" ", "_", regex=False)
)

# Renombrar columnas clave
wdi1 = wdi_raw.rename(columns={
    "economy": "countrycode",
    "time":    "year"
```

## 5.2 Tasa de crecimiento del PIB per cápita

::: {.panel-tabset}

## 🟦 R

```r
wdi2 <- wdi1 %>%
  arrange(countrycode, year) %>%
```

## 5.5 Urbanización vs Ingreso per cápita

Este gráfico muestra la relación entre el ingreso per cápita (en logaritmos) y la tasa de urbanización, comparando dos momentos: **1985** y **2021**. La curva verde = 1985, naranja = 2021.

::: {.panel-tabset}

## 🟦 R

```r
urban_income <- wdi1 %>%
  filter(year %in% c(1985, 2021)) %>%
  mutate(
    lngdppc = log(ny_gdp_pcap_kd),
    urban   = sp_urb_totl_in_zs
  ) %>%
  filter(!is.na(urban), !is.na(lngdppc))

hnd_1985 <- urban_income %>% filter(countrycode == "HND", year == 1985)
hnd_2021 <- urban_income %>% filter(countrycode == "HND", year == 2021)

ggplot(urban_income, aes(x = lngdppc, y = urban)) +
  geom_smooth(data = filter(urban_income, year == 1985),
              method = "loess", se = FALSE, color = "#1B7837", linewidth = 1.2) +
  geom_smooth(data = filter(urban_income, year == 2021),
              method = "loess", se = FALSE, color = "#D95F02", linewidth = 1.2) +
  geom_curve(
    aes(x = hnd_1985$lngdppc, y = hnd_1985$urban,
        xend = hnd_2021$lngdppc, yend = hnd_2021$urban),
    arrow = arrow(length = unit(0.2, "cm"), type = "closed"),
    curvature = 0.2, color = "#7B241C", linewidth = 1
  ) +
  geom_point(data = hnd_1985, aes(x = lngdppc, y = urban),
             size = 3.5, color = "#045A8D") +
  geom_text(data = hnd_1985, aes(label = "HND"),
            vjust = -1.1, hjust = 1.1, color = "#045A8D", size = 4.2) +
  annotate("text", x = 11.4, y = 23, label = "Trend 1985",
           color = "#1B7837", size = 4.2, fontface = "italic") +
  annotate("text", x = 11.4, y = 15, label = "Trend 2021",
           color = "#D95F02", size = 4.2, fontface = "italic") +
  labs(
    title    = "Cambios estructurales en Honduras (1985 → 2021)",
    subtitle = "Relación entre urbanización e ingreso per cápita (log USD)",
```

## 🐍 Python

```python
import numpy as np
from scipy.stats import loess  # o usamos lowess de statsmodels

from statsmodels.nonparametric.smoothers_lowess import lowess as sm_lowess

urban_income = (
    wdi1[wdi1["year"].isin([1985, 2021])]
    .assign(lngdppc = lambda df: np.log(df["gdp_pc"]))
    .dropna(subset=["urban_pct", "lngdppc"])
)

fig, ax = plt.subplots(figsize=(10, 6))

colores_anio = {1985: "#1B7837", 2021: "#D95F02"}

for anio, color in colores_anio.items():
    sub = urban_income[urban_income["year"] == anio].dropna()
    suavizado = sm_lowess(sub["urban_pct"], sub["lngdppc"], frac=0.5)
    ax.plot(suavizado[:, 0], suavizado[:, 1], color=color,
            linewidth=1.5, label=f"Tendencia {anio}")

# Honduras 1985 → 2021
hnd = urban_income[urban_income["countrycode"] == "HND"].sort_values("year")
ax.annotate(
    "", xy=(hnd.iloc[1]["lngdppc"], hnd.iloc[1]["urban_pct"]),
    xytext=(hnd.iloc[0]["lngdppc"], hnd.iloc[0]["urban_pct"]),
    arrowprops=dict(arrowstyle="->", color="#7B241C", lw=1.5)
)
ax.scatter(hnd.iloc[0]["lngdppc"], hnd.iloc[0]["urban_pct"],
           color="#045A8D", s=80, zorder=5)
ax.text(hnd.iloc[0]["lngdppc"] - 0.05, hnd.iloc[0]["urban_pct"] + 1.5,
        "HND", color="#045A8D", fontsize=11)

ax.set_title("Cambios estructurales en Honduras (1985 → 2021)",
             fontsize=15, fontweight="bold")
ax.set_xlabel("Ingreso per cápita (log USD constantes)")
ax.set_ylabel("Tasa de urbanización (%)")
ax.legend()
ax.grid(axis="y", alpha=0.3)
plt.tight_layout()
plt.savefig(graph / "urban_income.png", dpi=300)
plt.show()
```
